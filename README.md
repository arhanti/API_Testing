## 📌 Project Overview

This project demonstrates end-to-end API testing of the DummyJSON E-Commerce REST APIs using Postman and Newman.

The collection validates:

- Authentication APIs
- Product APIs
- Cart APIs
- CRUD Operations
- Response Validation
- Status Code Validation
- API Chaining
- Environment Variables
- Collection Variables
- Newman Automation
